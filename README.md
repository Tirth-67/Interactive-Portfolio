# 🚀 Creative Developer Portfolio

A modern, interactive portfolio website showcasing projects and skills with stunning animations and smooth interactions.

---

## ✨ Features

- 🎨 Modern dark theme with gradient accents
- 📱 Fully responsive design (desktop, tablet, mobile)
- 🗂️ Interactive project cards with modal details
- 📊 Animated skill progress bars
- 🔗 Smooth scroll navigation
- 📬 Contact form with email integration
- 🍔 Mobile-friendly hamburger menu

---

## 📁 Project Structure

```
portfolio/
├── index.html      # Main HTML structure
├── style.css       # Styling and animations
├── script.js       # Interactive functionality
└── README.md       # Documentation
```

---

## 🚀 Getting Started

1. Navigate to the project folder
2. Start a local server:

```bash
python -m http.server 8000
```

3. Open in browser: `http://localhost:8000`

## 📝 Customization

### Update Your Info
Edit these sections in `index.html`:
- Contact email (line 213)
- Location (line 219)
- Social links (lines 220–221)

### Change Colors
Edit CSS variables in `style.css` (lines 7–13):

```css
--primary-color: #667eea;
--secondary-color: #764ba2;
--accent-color: #00f2fe;
```

### Add Projects
Edit the `projectsData` array in `script.js` (line 1)
